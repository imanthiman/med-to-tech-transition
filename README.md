# Clinic Appointment Scheduler

A Django-based appointment management system designed for small medical practices.

## 🌟 Features
- Patient and Doctor registration/login
- Create, view, and manage appointments
- Admin panel for clinic staff
- Calendar view for upcoming bookings
- Email confirmation (optional stretch)

## 🛠️ Tech Stack
- Django 5.x
- Django REST Framework
- PostgreSQL (or SQLite for local dev)
- Docker (optional)
- GitHub Actions (CI/CD)

## 📅 Timeline
- **Week 4:** Project scaffold, user auth
- **Week 5:** Appointment models & views
- **Week 6:** Testing + deployment to Heroku/AWS

## 🔒 Stretch Goals
- Role-based access (doctor vs admin vs patient)
- JWT Authentication
- Email reminders via SendGrid
